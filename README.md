# 💍 Trường An & Ngọc Linh — Wedding Website

Trang web thiệp cưới cho lễ thành hôn của **Trường An & Ngọc Linh** ngày **29.03.2026**.

## 🚀 Chạy local

**Yêu cầu:** Node.js >= 18

```bash
# Cài đặt dependencies
npm install

# Chạy dev server
npm run dev
```

Truy cập tại: [http://localhost:3000](http://localhost:3000)

## 🏗️ Build production

```bash
npm run build
```

## 🛠️ Tech Stack

- **React 19** + **TypeScript**
- **Vite 6**
- **Tailwind CSS 4**
- **Motion** (animations)
